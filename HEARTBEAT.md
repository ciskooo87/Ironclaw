# HEARTBEAT.md

## Daily proactive check-ins for Paulo (America/Sao_Paulo)

If current Sao Paulo local time is within ±30 minutes of one of these slots, send a short check-in update:
- 09:00
- 13:00
- 18:00
- 22:00

Check-in format (short and objective):
1) Ironcore status now
2) Next 1-3 actions
3) Risks/blockers
4) Decision needed from Paulo (only if required)

Rules:
- Priority is always Ironcore.
- If Paulo appears asleep or inactive late night, avoid noisy pings; prepare concise summary for next useful slot.
- Confirm before any action with potential negative impact.
- If there is nothing meaningful to report, reply HEARTBEAT_OK.
